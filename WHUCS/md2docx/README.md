# 项目·缘由

> 前提：个人习惯使用 **md** 编写日志等；
> 
> 由于 WHUCS 有的实验报告要求提交 **word** 格式的文件，
> 以及经常需要不断反复调整标题、图片等文本字段的过程相当耗费精力；
> 
> 故而设计使用针对 docx 和 md 的 AST 分析库进行一个简易的转化，
> 项目整体结构和实现相当简单。
